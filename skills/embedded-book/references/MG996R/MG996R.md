# MG996R 金属舵机 180° 驱动参考

## 1. 基本参数

| 参数 | 值 |
|------|-----|
| 转动角度 | 0° ~ 180° |
| 工作电压 | 4.8V ~ 6.0V（推荐 5V） |
| 堵转扭矩 | ≈15 kg·cm（6V 供电） |
| 控制信号 | PWM，频率 **50Hz**（周期 **20ms**） |
| 脉冲范围 | **0.5ms ~ 2.5ms**（对应 0° ~ 180°） |
| 信号逻辑电平 | **3.3V / 5V 兼容** |

---

## 2. PWM 波形规范
* 周期20ms(50Hz)

脉宽|角度
|------|-----|
0.5ms|0°
1.0ms|45°
1.5ms|90°
2.0ms|135°
2.5ms|180°


### 角度-脉宽线性关系

每 1° 对应脉宽变化 ≈ **11.11 μs**。

$$\text{脉宽(μs)} = 500 + \frac{\text{角度} \times 2000}{180}$$

## 3. GPIO 能力需求

| 需求项 | 说明 |
|--------|------|
| GPIO 类型 | **PWM 输出**，需支持硬件定时器通道 |
| 逻辑电平 | 3.3V 或 5V 均可，GPIO 推挽输出 |
| 引脚数量 | **1个**（PWM 信号线） |
| 定时器资源 | 1个通用定时器（1 通道），**无需中断** |
| 其他引脚 | 无额外数字 IO，仅 PWM 信号线 |

> 每个舵机占用1个PWM通道。多舵机场景可使用同一定时器的多个通道或多路PWM扩展芯片（如 PCA9685）。

---

## 4. 电源需求

| 需求项 | 说明 |
|--------|------|
| 供电电压 | **4.8V ~ 6.0V DC**（推荐 5V） |
| 供电电流 | **≥ 2A**（堵转/重载峰值），轻载 ≈ 500~900mA |
| 供电方式 | **独立电源供电**，不可从 MCU 板载 3.3V/5V 引脚取电 |
| 共地要求 | 舵机 GND **必须**与 MCU 的 GND 连接（共地） |

> 供电不足会导致舵机无力、抖动或不转。

## 5. 接线定义

| 线色 | 信号 | 连接目标 |
|------|------|----------|
| 红色 | VCC (+) | 外部 5V 独立电源正极 |
| 棕色 | GND (-) | 外部电源负极 + MCU GND（共地） |
| 橙色 | PWM 信号 | MCU 定时器 PWM 输出引脚 |

## 6. 定时器 PWM 配置（平台通用）

### 高分辨率方案（推荐）

| 参数 | 要求 | 说明 |
|------|------|------|
| PWM 频率 | **50Hz** | 周期 20ms |
| 定时器时钟源 | ≥ 1MHz | 确保 ARR ≥ 20000 时频率为 50Hz |
| ARR（自动重载值） | **20000 - 1** | 20000 级分辨率 |
| CCR（比较值） | **500 ~ 2500** | 对应脉宽 0.5ms ~ 2.5ms |
| 角度精度 | ≈ 0.009° / step | — |

**配置公式（不同平台统一）：**

$$f_{PWM} = \frac{f_{TIM\_CLK}}{(PSC + 1) \times (ARR + 1)} = 50\ \text{Hz}$$

> 常见组合：令 $f_{TIM\_CLK} / (PSC + 1) = 1\ \text{MHz}$，则 ARR = 20000 - 1。
>
> - **ESP32**：ledcSetup，时钟源 80MHz，设置分频使计数器频率 = 1MHz，duty_resolution 需覆盖 20000 级（≥15 bit）。
> - **STM32**：PSC = (TIM_CLK / 1MHz) - 1，ARR = 20000 - 1。

### 低分辨率方案（省资源）

| 参数 | 值 |
|------|-----|
| ARR | 200 - 1 |
| CCR 范围 | 5 ~ 25 |
| 精度 | ≈ 0.9° / step |

---

## 7. 角度-CCR 换算公式

$$CCR = \frac{angle \times (CCR_{max} - CCR_{min})}{180} + CCR_{min}$$

高分辨率方案（CCR 500~2500）：

$$CCR = \frac{angle}{9} + 500$$

## 9. 驱动要点

1. **上电初始化**：输出0°位置脉宽（0.5ms），避免舵机突然动作。
2. **控制周期**：推荐 **20ms**（与 PWM 周期一致），每个周期更新一次 CCR。
3. **平滑过渡**：避免角度跳变导致抖动，使用增量步进或 PID 控制使角度平滑变化。
4. **角度限幅**：输出角度需钳位在 0°~180° 范围内，超出可能损坏机械结构。
5. **堵转保护**：到达目标后不再持续修正，避免长时间堵转发热。
6. **多舵机**：多个舵机共用一个定时器的不同通道即可，无需额外定时器。
7. **中断驱动**（可选）：使用定时中断（20ms）自动更新舵机位置，主循环处理其他任务。

---

## 10. 常见故障排查

| 现象 | 可能原因 | 解决 |
|------|----------|------|
| 舵机不转 | 供电不足 / 未调 PWM 输出 | 使用独立 5V/2A 电源；检查 PWM 输出使能 |
| 角度不准 | 脉宽偏离标称值 | 微调 CCR 最小/最大值校准 |
| 剧烈抖动 | 目标角度突变 | 启用平滑控制或减小步长 |
| 舵机发热 | 长时间堵转 | 确保角度在行程内，到达后停止驱动 |
| PWM 无输出 | 定时器/GPO 未初始化 | 检查 PWM 通道配置和 GPIO 复用 |
