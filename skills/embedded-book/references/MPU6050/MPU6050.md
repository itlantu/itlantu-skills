# MPU6050 六轴姿态传感器 驱动参考

## 1. 基本参数

| 参数 | 值 |
|------|-----|
| 芯片型号 | MPU-6050 (InvenSense / TDK) |
| 模块型号 | GY-521 (常用 breakout 板) |
| 传感器类型 | 三轴加速度计 + 三轴陀螺仪 + 温度传感器 |
| 工作电压 | **2.375V ~ 3.46V**（芯片），GY-521 模块板载 LDO 支持 **3.3V ~ 5V** |
| 工作电流 | 陀螺仪 + 加速度计全速运行：**3.9mA**（典型值），待机：**5μA** |
| 通信接口 | **I2C**（最高 400kHz Fast Mode） |
| 工作温度 | -40°C ~ +85°C |
| 物理尺寸 | GY-521 模块：20mm × 16mm × 1.6mm（PCB） |
| 加速度计量程 | ±2g / ±4g / ±8g / ±16g（可配置） |
| 陀螺仪量程 | ±250 / ±500 / ±1000 / ±2000 °/s（可配置） |
| 加速度计输出 | 16 位 ADC（有符号） |
| 陀螺仪输出 | 16 位 ADC（有符号） |
| 温度输出 | 16 位有符号，范围 -40°C ~ +85°C |
| 片上 DMP | 内置数字运动处理器（Digital Motion Processor），可解算四元数 |

---

## 2. GPIO 能力需求

| 需求项 | 说明 |
|--------|------|
| GPIO 类型 | I2C（开漏，需外部上拉电阻） |
| 逻辑电平 | **3.3V**（GY-521 模块板载 LDO 及电平适配，可接 5V MCU） |
| 引脚数量 | **2 个**（SCL + SDA），可选 INT 中断引脚 **1 个** |
| 外设资源 | 1 路 I2C 控制器（硬件 I2C 或软件模拟 I2C） |
| 特殊需求 | SDA/SCL 需 **2.2kΩ ~ 4.7kΩ 上拉电阻**（GY-521 模块已板载）；INT 引脚为开漏输出，需外部上拉 |

> GY-521 模块上 SDA/SCL 已含板载上拉电阻。AD0 引脚默认拉低（GND），I2C 地址为 0x68。

---

## 3. 电源需求

| 需求项 | 说明 |
|--------|------|
| 供电电压 | GY-521 模块：**3.3V ~ 5V DC**（VCC 引脚经板载 LDO 降压至 3.3V 供芯片） |
| 供电电流 | 典型 **3.9mA**（全功能运行），峰值 < 10mA |
| 供电方式 | **MCU 引脚供电即可**（电流小，可直接从 3.3V/5V 引脚取电） |
| 共地要求 | 模块 GND **必须**与 MCU 的 GND 连接（共地） |

> GY-521 模块功耗极低，直接从 MCU 开发板的 3.3V 或 5V 电源引脚供电即可，无需外部独立电源。

---

## 4. 接线定义

GY-521 模块引脚（8 Pin 排针）：

| 序号 | 引脚名 | 功能 | 连接目标 | 备注 |
|------|--------|------|----------|------|
| 1 | VCC | 电源正极 | MCU 3.3V 或 5V | 板载 LDO，3.3V~5V 均可 |
| 2 | GND | 电源地 | MCU GND | 必须共地 |
| 3 | SCL | I2C 时钟线 | MCU SCL (I2C) | 已板载上拉电阻 |
| 4 | SDA | I2C 数据线 | MCU SDA (I2C) | 已板载上拉电阻 |
| 5 | XDA | 辅助 I2C 数据 | 外部磁力计传感器（可选） | 一般悬空 |
| 6 | XCL | 辅助 I2C 时钟 | 外部磁力计传感器（可选） | 一般悬空 |
| 7 | AD0 | I2C 地址选择 | GND（地址 0x68）或 VCC（地址 0x69） | 默认接 GND |
| 8 | INT | 中断输出 | MCU 任意 GPIO（可选） | 开漏输出，需上拉 |

## 5. 通信协议

| 参数 | 值 |
|------|-----|
| 协议类型 | **I2C** |
| 从机地址 | **0x68**（7-bit，AD0=GND）；**0x69**（7-bit，AD0=VCC） |
| 写入地址字节 | 0xD0（0x68 << 1） |
| 读取地址字节 | 0xD1（(0x68 << 1) \| 1） |
| 时钟速率 | 标准模式 100kHz / 快速模式 **400kHz** |
| 数据格式 | **大端**（高字节在前），有符号 16 位 |

## 6. 寄存器说明

### 6.1 关键配置寄存器

| 寄存器地址 | 名称 | 读写 | 位宽 | 描述 |
|------------|------|------|------|------|
| 0x6B | PWR_MGMT_1 | R/W | 8 | 电源管理 1：复位、睡眠、时钟源选择 |
| 0x6C | PWR_MGMT_2 | R/W | 8 | 电源管理 2：各轴待机控制 |
| 0x19 | SMPLRT_DIV | R/W | 8 | 采样率分频器 |
| 0x1A | CONFIG | R/W | 8 | 外部帧同步 + DLPF 配置 |
| 0x1B | GYRO_CONFIG | R/W | 8 | 陀螺仪自检 + 满量程选择 |
| 0x1C | ACCEL_CONFIG | R/W | 8 | 加速度计自检 + 满量程选择 + 高通滤波 |
| 0x75 | WHO_AM_I | R | 8 | 器件 ID 寄存器，复位值 **0x68** |

### 6.2 PWR_MGMT_1 (0x6B) 位定义

| 位 | 名称 | 描述 |
|----|------|------|
| [7] | DEVICE_RESET | 写 1 复位所有寄存器（自动清零） |
| [6] | SLEEP | 1 = 进入睡眠模式 |
| [5] | CYCLE | 1 = 循环模式（睡眠与唤醒交替） |
| [4] | — | 保留 |
| [3] | TEMP_DIS | 1 = 禁用温度传感器 |
| [2:0] | CLKSEL | 时钟源选择（见下表） |

**CLKSEL 时钟源：**

| CLKSEL | 时钟源 |
|--------|--------|
| 0 | 内部 8MHz 振荡器（默认） |
| 1 | PLL X 轴陀螺仪参考（**推荐**） |
| 2 | PLL Y 轴陀螺仪参考 |
| 3 | PLL Z 轴陀螺仪参考 |
| 4 | 外部 32.768kHz |
| 5 | 外部 19.2MHz |
| 6 | 保留 |
| 7 | 停止时钟并复位 |

> **上电初始化**：向 PWR_MGMT_1 写入 **0x00** 唤醒设备（清除 SLEEP 位），或写入 **0x01** 选择 PLL X 陀螺仪为时钟源。

### 6.3 GYRO_CONFIG (0x1B) — 陀螺仪量程

| FS_SEL [4:3] | 满量程 | 灵敏度 (LSB/°/s) |
|--------------|--------|-------------------|
| 0 | **±250 °/s** | 131 |
| 1 | ±500 °/s | 65.5 |
| 2 | ±1000 °/s | 32.8 |
| 3 | ±2000 °/s | **16.4** |

> 默认值：0x00（±250 °/s）。设置时保持 Bit[2:0] 为 0，避免使能自检。

### 6.4 ACCEL_CONFIG (0x1C) — 加速度计量程

| AFS_SEL [4:3] | 满量程 | 灵敏度 (LSB/g) |
|---------------|--------|-----------------|
| 0 | **±2g** | 16384 |
| 1 | ±4g | 8192 |
| 2 | ±8g | 4096 |
| 3 | ±16g | **2048** |

> 默认值：0x00（±2g）。Bit [2:0] 控制加速度计高通滤波器。

### 6.5 CONFIG (0x1A) — DLPF 设置

| DLPF_CFG [2:0] | 加速度计带宽 | 加速度计延迟 | 陀螺仪带宽 | 陀螺仪延迟 | 采样率 |
|-----------------|-------------|-------------|-----------|-----------|--------|
| 0 | 260Hz | 0ms | 256Hz | 0.98ms | **8kHz** |
| 1 | 184Hz | 2.0ms | 188Hz | 1.9ms | 1kHz |
| 2 | 94Hz | 3.0ms | 98Hz | 2.8ms | 1kHz |
| 3 | 44Hz | 4.9ms | 42Hz | 4.8ms | 1kHz |
| 4 | 21Hz | 8.5ms | 20Hz | 8.3ms | 1kHz |
| 5 | 10Hz | 13.8ms | 10Hz | 13.4ms | 1kHz |
| 6 | **5Hz** | 19.0ms | 5Hz | 18.6ms | 1kHz |
| 7 | 保留 | — | 保留 | — | 保留 |

> **采样率公式**：$Sample Rate = \frac{Gyroscope\ Output\ Rate}{1 + SMPLRT\_DIV}$
>
> DLPF_CFG = 0 时，陀螺仪输出率为 8kHz；DLPF_CFG = 1~6 时，陀螺仪输出率为 1kHz。

### 6.6 数据输出寄存器

| 起始地址 | 数据 | 长度 | 格式 |
|----------|------|------|------|
| 0x3B | ACCEL_XOUT | 2 字节 | 有符号 16-bit，大端 |
| 0x3D | ACCEL_YOUT | 2 字节 | 有符号 16-bit，大端 |
| 0x3F | ACCEL_ZOUT | 2 字节 | 有符号 16-bit，大端 |
| 0x41 | TEMP_OUT | 2 字节 | 有符号 16-bit，大端 |
| 0x43 | GYRO_XOUT | 2 字节 | 有符号 16-bit，大端 |
| 0x45 | GYRO_YOUT | 2 字节 | 有符号 16-bit，大端 |
| 0x47 | GYRO_ZOUT | 2 字节 | 有符号 16-bit，大端 |

> 可从 0x3B 连续读取 14 字节，一次性获取加速度计、温度、陀螺仪的 7 组数据。

## 7. 数据解析与换算

### 7.1 加速度计

从原始值到物理量（以 g 为单位）：

$$\text{加速度}(g) = \frac{Raw}{LSB}$$

| 满量程 | LSB/g | 换算 |
|--------|-------|------|
| ±2g | 16384 | $accel = raw / 16384.0$ |
| ±4g | 8192 | $accel = raw / 8192.0$ |
| ±8g | 4096 | $accel = raw / 4096.0$ |
| ±16g | 2048 | $accel = raw / 2048.0$ |

### 7.2 陀螺仪

从原始值到物理量（以 °/s 为单位）：

$$\text{角速度}(°/s) = \frac{Raw}{LSB}$$

| 满量程 | LSB/(°/s) | 换算 |
|--------|-----------|------|
| ±250°/s | 131 | $gyro = raw / 131.0$ |
| ±500°/s | 65.5 | $gyro = raw / 65.5$ |
| ±1000°/s | 32.8 | $gyro = raw / 32.8$ |
| ±2000°/s | 16.4 | $gyro = raw / 16.4$ |

### 7.3 温度

$$\text{温度}(°C) = \frac{Raw}{340} + 36.53$$

> 近似公式：$Temperature = 35 + \frac{Raw + 13200}{280}$

## 8. 数据读取流程

### 8.1 初始化流程

1. 配置 I2C 总线，设置时钟速率 100kHz（标准）或 400kHz（快速）。
2. 上电等待模块稳定（建议延时 ≥ 50ms）。
3. **验证连接**：读取 WHO_AM_I 寄存器 (0x75)，确认返回 **0x68**（或 0b110100）。
4. **唤醒设备**：向 PWR_MGMT_1 (0x6B) 写入 **0x00**（清除 SLEEP 位，使用内部时钟），或写入 **0x01**（PLL X 陀螺仪参考时钟，推荐）。
5. **配置量程**：向 GYRO_CONFIG (0x1B) 写入目标量程值，向 ACCEL_CONFIG (0x1C) 写入目标量程值。
6. **配置 DLPF**：向 CONFIG (0x1A) 写入 DLPF 值（推荐 0x06 = 5Hz 带宽，滤波效果较好）。
7. **配置采样率**：向 SMPLRT_DIV (0x19) 写入分频值。

### 8.2 读取数据流程

1. 向 I2C 总线发送器件写入地址 (0xD0)，然后发送起始寄存器地址 (0x3B)。
2. 发送 I2C 重复起始信号。
3. 向 I2C 总线发送器件读取地址 (0xD1)。
4. 连续读取 14 字节数据（自动地址递增）。
5. 大端拼接：$Data = (HighByte << 8) | LowByte$。
6. 按量程换算为物理量。

### 8.3 典型初始化示例（平台无关）

```
1. 延时 ≥ 50ms（上电稳定）
2. 写 PWR_MGMT_1   ← 0x01    // 唤醒 + PLL X 陀螺仪参考时钟
3. 写 SMPLRT_DIV    ← 0x07    // 采样率 = 1kHz / (1+7) = 125Hz
4. 写 CONFIG        ← 0x06    // DLPF_CFG = 6，带宽 5Hz
5. 写 GYRO_CONFIG   ← 0x00    // 陀螺仪 ±250 °/s
6. 写 ACCEL_CONFIG  ← 0x00    // 加速度计 ±2g
```

## 9. 校准参数

| 参数 | 说明 |
|------|------|
| 加速度计偏移 | 寄存器 0x06~0x0B（XA_OFFS / YA_OFFS / ZA_OFFS），上电后从 OTP 自动加载 |
| 陀螺仪偏移 | 寄存器 0x13~0x18（XG_OFFS_USR / YG_OFFS_USR / ZG_OFFS_USR），需手动校准写入 |
| 自检 | GYRO_CONFIG 和 ACCEL_CONFIG 的 Bit[7:5] 可启用出厂自检（验证传感器功能） |

### 陀螺仪零偏校准方法

1. 将模块静置于水平面上。
2. 连续读取陀螺仪原始数据（如 100~500 次），计算各轴平均值。
3. 将平均值取反后乘以 -1，写入对应的 OFFS_USR 寄存器。
4. 校准后静置读数应接近 0。

> GY-521 模块出厂时陀螺仪零偏通常不为零，建议上电后执行零偏校准。

## 10. 驱动要点

1. **上电初始化**：必须先唤醒设备（写 PWR_MGMT_1 = 0x01），否则传感器处于睡眠状态、输出全为固定值。
2. **时钟源选择**：推荐使用 PLL X 陀螺仪参考时钟（CLKSEL=1），比内部振荡器更精确、抖动更小。
3. **数据读取**：推荐使用 I2C 连续读取（从 0x3B 起 14 字节），一次 I2C 事务获取全部传感器数据，避免多次单字节读取的时序漂移。
4. **数据溢出**：加速度计和陀螺仪为 16 位有符号整数，超出量程时数据饱和（±32767），不会回绕，但此时数据已失真。
5. **DLPF 滤波**：DLPF 值越小（带宽越低），噪声越小但延迟越大。姿态解算推荐 DLPF_CFG = 3（42Hz）或 4（20Hz），平衡噪声与响应速度。
6. **中断驱动读取**：可通过 INT 引脚和 INT_ENABLE 寄存器使能"数据就绪中断"，在中断服务程序中读取数据，减少 I2C 轮询开销。
7. **同一总线多设备**：通过 AD0 引脚可选两个地址（0x68 / 0x69），同一 I2C 总线最多挂 2 个 MPU6050。如需更多设备，需使用 I2C 多路复用芯片。
8. **DMP 使用**：芯片内置 DMP（数字运动处理器），可硬件解算四元数姿态角。需加载 DMP 固件到芯片内存（约 3KB），然后从 FIFO 读取四元数数据。DMP 输出速率可配置。

## 11. 常见故障排查

| 现象 | 可能原因 | 解决方法 |
|------|----------|----------|
| 读取 WHO_AM_I 失败或返回 0xFF | I2C 接线错误 / 地址不对 / 上拉电阻缺失 | 检查 SDA-SCL 接线；确认模块供电正常；检查 AD0 引脚电平确定地址 |
| WHO_AM_I 返回非 0x68 | 模块损坏或非 MPU6050 | 确认模块型号，MPU6050 应返回 0x68（低 6 位为 0b110100） |
| 数据始终为 0 或不变 | 传感器处于睡眠状态 | 写 PWR_MGMT_1 = 0x00 或 0x01 唤醒设备 |
| 数据波动/噪声大 | DLPF 带宽过高、电源纹波 | 降低 DLPF_CFG 值（如设为 5Hz），检查电源去耦电容 |
| 加速度数据有较大零偏 | 未校准或安装倾斜 | 执行水平校准；在安装平台上读取基准值作为零偏 |
| 陀螺仪静置时读数不为零 | 未做零偏校准 | 执行陀螺仪零偏校准流程（见第 9 节） |
| I2C 通信不稳定 / 偶发错误 | 线缆过长、速率过高、上拉电阻过大 | 降低 I2C 速率至 100kHz；缩短线缆（< 20cm）；确认上拉电阻 2.2kΩ~4.7kΩ |
| 模块发热严重 | VCC 接入过高电压、短路 | 确认供电 ≤ 5V；检查 GY-521 模块是否有短路 |
